# delivery_bot 🤖

Телеграмм бот для доставки еды с корзиной и меню, оформлением заказа.

## 🚀 Функционал
- `/start` — приветствие
- `/add_info` - ввод информации
- Инлайн-клавиатура меню ресторана
- Инлайн-клавиатура корзины
- Реплай-клавиатура меню бота

## 🛠 Технологии
- Python 3.9.13
- pyTelegramBotAPI (telebot)
- dotenv

## 📦 Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone git@github.com:varvar-var/delivery_bot.git
   cd delivery_bot
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` в корневой папке и добавьте токен:
   ```
   BOT_TOKEN=ваш_токен_от_BotFather
   ```

4. Запустите бота:
   ```bash
   python bot.py
   ```
